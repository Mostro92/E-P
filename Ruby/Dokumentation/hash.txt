Ein Hash ist ein assoziatives Array, das heißt ein Array mit einem selbst definierbaren index.

cars ={'eins' => 'Audi', 'zwei' => 'BMW'}

Der Entwickler von Ruby hat aus Pervormance Gründen für den Hash -Index das Symbol vorgesehen:

cars ={:eins => 'Audi', :zwei => 'BMW'}

Kurzschreibweise
cars ={eins: 'Audi', zwei:'BMW'}


Iteration über einen Hash:
	cars.each do |key, val|
		puts "Index: #{key} ist ein #{val}"
	end

people.values.sort    # => [18, 23, 54]

people.sort_by { |name, age| age }
